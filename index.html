Project Overview

This is a comprehensive 3D helicopter model created as a university project using 3ds Max. The project showcases the detailed modeling process from concept sketches and reference images to a fully rendered 3D model.

Project Components

  • 3D Model: Detailed helicopter mesh with all major components
